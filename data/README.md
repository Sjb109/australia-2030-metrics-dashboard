This folder holds the raw time‑series data used by the dashboard.  Each
metric is stored as a CSV file named `metric_id.csv` containing columns:

* `date` – the observation date (datetime)
* `value` – the observed value (decimal fraction; e.g. 0.035 for 3.5 %)
* `series_key` – identifier provided by the source API (when applicable)
* `target` – the 2030 target value (repeated for each row for convenience)

The files are automatically generated by `scripts/update_data.py`.  They are
version controlled to allow tracking changes over time.
